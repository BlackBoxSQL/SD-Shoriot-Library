# 1. Python কি?

 	* python, একটি জনপ্রিয় প্রোগ্রামিং ল্যাঙ্গুয়েজ। 1991 সালে Guido van Rossum  এটি  তৈরি করেন।

 	* python, একটি object-oriented high level language.

 	* python, একটি general-purpose language. অর্থাৎ প্রোগ্রামিং ল্যাঙ্গুয়েজ দিয়ে করা যায় এ রকম প্রায় সবকিছুই python ইউস করে করা যায়।

 	* python, একটি হাই-লেভেল, ইন্টারপ্রিটেড, ইন্টারেক্টিভ ও অবজেক্ট-অরিয়েন্টেড স্ক্রিপ্টিং ল্যাঙ্গুয়েজ এবং python একটি ডাইনামিক ল্যাঙ্গুয়েজও বটে।
